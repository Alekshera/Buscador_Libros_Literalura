
*Buscador de Libros LITERALURA


*[Descripción del proyecto]

Proyecto propuesto por Alura y ONE en el que se realiza un buscador de libros por titulo, estos quedan grabados en la base de datos, en el que se pueden consultar todos los libros buscados y tambien los autores,
se pueden encontrar por años los autores que estaban vivos y hacer busquedas de los libros buscados por idioma.

*[Estado del proyecto]

El proyecto esta terminado con las funcionalidades básicas, en el futuro se desea integrar más APIs que funcionen en la misma aplicación, como una de traducción, otra de imágenes y de datos estadisticos, 
se desea hacer una interfaz de usuario interactiva utilizando react.
